# Odin-Recipes

Odin-Recipes is the **first project** from *The Odin Project – HTML & CSS Foundations*.  
The goal of this project is to practice building a basic website structure using HTML and styling it with CSS .

---

## Project Structure
- **Home Page**: Contains links to all recipes
- **3 Recipe Pages**: Each page displays a different food recipe with its ingredients and steps
